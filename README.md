# CipherFile

Bu proje C++ ile geliştirilmiş XOR tabanlı dosya şifreleme/çözme uygulamasıdır.

## Özellikler
- Dosya şifreleme
- Dosya çözme
- Basit konsol arayüzü

## Kullanım
1. Kodları bilgisayarına çek.
2. Derleyici ile main.cpp / dene2.cpp
   g++ dene2.cpp -o CipherFile.exe
3. Programı çalıştır:
   ./CipherFile.exe

## Açıklama
Bu uygulama bir dosyayı XOR algoritmasıyla byte byte şifreler veya çözer.
